# ZOMATO-RESTAURANTS-ANALYSIS
# RESTAURANTS DATA ANALYSIS USING GOOGLE SHEETS

## Objective
Analyze Zomato restaurant data to identify trends in ratings, pricing, locations, and online delivery impact.

## Tools Used
- Google Sheets
- Pivot Tables
- Charts & Filters

## Dataset
Zomato restaurant dataset including:
- Restaurant name
- City
- Rating
- Votes
- Online delivery
- Price range

## Analysis Performed
- City-wise restaurant distribution
- Rating analysis
- Online delivery vs rating comparison
- Price range vs customer preference

## Key Insights
- Mid-priced restaurants dominate the market
- Restaurants offering online delivery tend to have better ratings
- A few cities account for majority of listings

## Files in this Repository
- `zomato_data.csv` – Raw dataset
- `zomato_analysis.pdf` – Complete analysis with charts and insights

## Conclusion
This analysis provides actionable insights into customer behavior and restaurant performance on food delivery platforms.
